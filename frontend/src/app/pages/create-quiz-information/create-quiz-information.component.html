Informations sur la création d'un quiz

A ce niveau vous pouvez mettre jusqu'à 3 images pour chaque question, elles seront ajoutées sucessivement à chaque erreur.

Au bout de 3 erreurs le quiz passe à la question suivante.

Pour ajouter des questions au quiz, il faudra aller dans la rubrique "modifier les quiz" une fois le quiz crée.
