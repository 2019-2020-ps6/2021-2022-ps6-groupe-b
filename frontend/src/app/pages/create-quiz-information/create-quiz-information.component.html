Informations sur la création d'un quiz

Vous allez être amené à créer un quiz. Celui-ci sera automatiquement adapté au stade de la maladie du joueur. Pour créer un quiz, il est indispensable d’ajouter des URL d’images que vous pouvez récupérer sur votre cloud par exemple. Pour cela, il suffit de faire un clic droit sur l’image, puis cliquer sur le bouton “copier l’adresse de l’image”. Vous allez aussi être invité à saisir la question à poser, plusieurs réponses et le lien de parenté si vous avez choisi un quiz sur les personnes.

Une fois le bouton validé cliqué, vous serez redirigé vers une page pour ajouter les questions du quiz.
