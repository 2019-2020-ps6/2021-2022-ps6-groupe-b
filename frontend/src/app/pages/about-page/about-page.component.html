À propos ...

Ce site web a pour vocation à faire passer un moment agréable aux personnes âgées en se remémorant les bons souvenirs de sa vie !

Comment créer un quiz

Pour créer un quiz, il suffit d'être administrateur et de cliquer sur gérer les quiz puis cliquer sur créer un quiz et enfin suivre les instructions

Une fois le quiz rempli, celui-ci sera automatiquement adapté au stade de la maladie du joueur.

Comment jouer

Pour jouer il suffit de cliquer sur "Jouer" puis de choisir une categorie et de sélectionner un quiz. Une fois le quiz sélectionné, il suffit de cliquer sur "Jouer" pour commencer le quiz.

Le quiz est composé de questions et de réponses. Il est possible de répondre à une question en cliquant sur la réponse qui correspond à la bonne réponse.

Comment voir les statistiques

Il faut aller dans gérer les quiz, puis statistiques, depuis cette page vous verrez une statistique globale qui indique quelle personne est cochée correctement avec un pourcentage, puis vous pouvez aussi voir essai par essai chaque quiz pour voir quelles images pourraient poser problème
